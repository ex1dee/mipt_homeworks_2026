**Модель**: `GigaChat`

---

#### Фичи:

- Переключение между чатами
- Сохранение контекста каждого чата в файл
- Потоковый вывод сообщения
- Цветной текст в консоли
- Динамическое именование: если чат создан без указания наименования, LLM после первого сообщения сама назовёт чат

---

#### Управление контекстом

- Системный промпт в файле проекта
- Автоматическое очищение старых сообщений, не входящих более в контексное окно
- Автоматическое сохранение контекста текущего чата

---

#### Команды:

- `/new` - Создать чат с названием "Новый чат №<ID>"
- `/new <Имя>` - Создать чат с указанным именем
- `/rename <Новое имя>` - Изменение имени текущего чата
- `/rename <ID> <Новое имя>` - Изменение имени чата с указанным ID
- `/switch <ID>` - Переключиться на чат с указанным ID
- `/delete <ID>` - Удалить чат с указанным ID
- `/delete` - Удалить текущий чат (после чего автоматически создаётся новый)
- `/clear` - Очистка контекста текущего чата
- `/list` - Вывод списка чатов (с пагинацией)
- `/exit` - Выйти из приложения
